= DuckDB Installation
:toc:
:toclevels: 2

== Offizielle Installationsseite
Besucht die offizielle Installationsseite von DuckDB unter folgendem Link:  
https://duckdb.org/install/

Die Seite erkennt automatisch euer Betriebssystem (z. B. **Windows**, **macOS** oder **Linux**) und zeigt die passenden Installationsanweisungen an.

== Alternativen zur Installation

=== Python
[source,bash]
----
pip install duckdb
----

=== Node.js
[source,bash]
----
npm install @duckdb/node-api
----

=== macOS (Homebrew)
[source,bash]
----
brew install duckdb
----

=== macOS & Linux (CURL)
[source,bash]
----
curl https://install.duckdb.org | sh
----

== Starten von DuckDB

Nach der Installation kann DuckDB direkt in der Kommandozeile gestartet werden:
[source,bash]
----
duckdb
----

== DuckDB Benutzeroberfläche (UI)

Mit folgendem Befehl öffnet ihr die grafische Oberfläche im Browser:
[source,bash]
----
duckdb -ui
----

In dieser **UI** könnt ihr SQL-Befehle ausführen und eure Datenbanken verwalten.

=== Neues Notebook erstellen
Klickt auf das **Plus-Symbol neben „Notebooks“**, um ein neues Notebook zu erstellen.  
→ Funktioniert ähnlich wie **Jupyter Notebooks** oder **Google Colab**.

=== Datenbank verbinden oder erstellen
Klickt auf das **Plus-Symbol neben „Attached Databases“**, um eine neue Datenbank zu erstellen oder eine bestehende zu öffnen.

* **Location:** `File`  
* **Path:** Pfad zu eurer DuckDB-Datei  
  z. B. `C:/Users/Name/Desktop/qisinsights/students_clean.duckdb`  
* **Alias:** kann leer bleiben

== Ausführen von SQL-Abfragen

In einer Notebook-Zelle könnt ihr SQL-Befehle direkt eingeben und mit *Run* ausführen.  
Beachtet: Der vollständige Tabellenpfad muss das Schema `datenbankname.main.tabellenname` enthalten.

=== Beispiele

[source,sql]
----
-- Zeigt alle Einträge der Tabelle students
SELECT * FROM students_final.main.students;

-- Zeigt alle Einträge der Tabelle studienrichtung
SELECT * FROM students_final.main.studienrichtung;

-- Zeigt alle Einträge im View stg_students_clean
SELECT * FROM students_final.main.stg_students_clean;

-- Zeigt alle Einträge im View stg_students_with_credits
SELECT * FROM students_final.main.stg_students_with_credits;
----

TIP: Die UI zeigt euch automatisch diverse Informationen an, z. B. die **Anzahl der Zeilen**, die **Distinct-Werte** einer Spalte oder **Statistiken** (z.B. **Mittelwert**, **Median**, **Maximum**, **Minimum** und **Standardabweichung** und Quantile) zu numerischen Spalten.
